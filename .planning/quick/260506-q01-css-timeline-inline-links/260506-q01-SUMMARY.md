## Verification

`yarn build` exits 0 with no TypeScript errors.

## How to add more links

In any `eras` entry, change a `milestones` string to an object:
```ts
{
  text: "The original milestone string goes here with word to link",
  links: {
    "word to link": "https://example.com",
    "another word": "https://another.com",
  },
}
```
Multiple linked words per milestone are supported.
