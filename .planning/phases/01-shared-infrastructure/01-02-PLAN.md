<objective>
Remove the `@media (prefers-color-scheme: dark)` block from `src/app/globals.css` so the class-based `@custom-variant dark` is the single dark-mode source of truth. Per D-10 and INFRA-03 — `ThemeToggle` controls dark mode; OS preference must not compete.

Purpose: With both mechanisms active, a user on a dark-OS who clicks ThemeToggle to "light" sees the OS-driven `--background: #0a0a0a` override the toggle, producing visual mismatches between OS-light and OS-dark users for the same toggle state. Removing the media query block makes ThemeToggle authoritative.

Output: `src/app/globals.css` with the prefers-color-scheme block deleted; all other content preserved.

This plan is independent of Plan 01 (no package dependency — pure CSS edit) so it runs in Wave 1 in parallel.
</objective>
